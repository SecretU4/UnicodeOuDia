3.1.　メインウインドウ

　中央のバーで仕切られた左側を『路線ビュー』と呼びます。初期状態では、『路線ビュー』には、[路線]・[駅]・[列車種別]・[ダイヤ]の４つのアイコンがあります。
バーの右側の領域を『作業領域』と呼びます。『時刻表』や『ダイヤグラム』は、この『作業領域』に表示されます。

参照：『基本的な使い方』の『2.1.1.1. OuDiaを起動』

3.1.1.　メニューコマンド

メインウインドウのメニューは、『作業領域』でアクティブになっている（最前面にある）ビューの種類によって変化します。
ここでは、どのビューでも共通のメニューコマンドについてのみ説明します。
ビューに特有のメニューコマンドは、それぞれのビューの項で説明します。

  1. [ファイル]
  2. [表示]
  3. [ウインドウ]
  4. [ヘルプ]
